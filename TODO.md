# Minimum features
- Vulkan renderer with bindless
- GPU driven culling
- Raytraced shadows
- CSM
- Simple post processing
- Skybox rendering
- IBL
- C++ scripting (xmake compilation)
- Physics (jolt)
- Audio (miniaudio)
- Editor/serialization/scenes blablabla (ImGui)
- Actor model for entities
- GLTF model loading and image loading with STB
- Simple asset export pipeline (BC7, compressed mesh format)
- Simple in-game UI

## Would be good to have

- Animations
- LOD selection
- Mesh shaders
- SSR
- RTAO
- AngelScript support
